# hotels
Приложение для бронирования отелей. 

# server
Сервер rest api. Пишется на следующем стеке:
1. node.js
2. express.js
3. express-validator
В планах брать данные из Mysql и использовать Sequelize ORM

# client
Клиент для приложения. Пишется на стеке:
1. Vue.js
2. Vue-router
3. Vuex
4. Vuetify
